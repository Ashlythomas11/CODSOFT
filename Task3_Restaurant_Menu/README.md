# Task 3 – Restaurant Menu UI/UX

## CodSoft UI/UX Internship

### Project Overview
This project is a modern Restaurant Menu mobile application designed using Figma. The interface focuses on simplicity, usability, and an elegant dining experience.

### Screens Included
- Splash Screen
- Welcome Screen
- Menu Screen
- Food Details Screen

### Design Features
- Clean and modern interface
- Green & cream color palette
- Easy navigation
- Categorized food menu
- Food cards with ratings and pricing
- Responsive mobile layout
- Interactive prototype
