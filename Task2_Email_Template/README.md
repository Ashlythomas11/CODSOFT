# 🌿 Task 2 – Email Template UI Design

## 📌 Project Overview
This project was completed as part of my **CodSoft UI/UX Design Internship**.

I designed a modern and responsive email newsletter for **Plantify**, a plant care application. The email template focuses on a clean layout, consistent branding, and an engaging user experience while following professional email design principles.

---

## 🎯 Objective
Design a visually appealing email template that:
- Looks clean and professional
- Is easy to read and navigate
- Uses a consistent color palette
- Includes clear call-to-action buttons
- Follows responsive email design principles

---

## ✨ Features
- 🌿 Plantify branding
- Modern hero section
- Weekly Plant Care Tips section
- Special Offer promotional banner
- Featured Plant section
- Call-to-Action buttons
- Professional footer
- Clean and minimal green theme

---
